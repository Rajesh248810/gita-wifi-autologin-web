Installation Guide

Choose your platform and follow the steps below.

1

Download & Extract

Download the ZIP and extract it (Right-click → Extract All...) to your Desktop or Documents.

2

Run Setup

Double-click setup.bat and grant Administrator access. The script registers as a Windows Task.

3

Done!

Close any Sophos login pages. You're connected silently in the background forever.

🗑️

To uninstall: double-click uninstall.bat (requires Admin).

💡 No ZIP needed on Android — just follow these steps.
1

Download MacroDroid

Install the free MacroDroid app from Google Play Store.

2

Create Trigger

Trigger: Connectivity → Wi-Fi Connection → Connects → select GITA Wi-Fi SSID.

3

Set HTTP Action

Method: POST  |  URL: http://20.2.0.1:8090/login.xml
Body: mode=191&username=ID&password=PASS&producttype=2

4

Add Heartbeat

Every 30s: HTTP GET to http://20.2.0.1:8090/live?mode=192&username=ID&producttype=2

💡 No ZIP needed on iPhone — use the built-in Shortcuts app.
1

Open Shortcuts App

Launch the built-in Shortcuts app on your iPhone.

2

Create Automation

Tap Automation → + → Personal Automation → Wi-Fi → select GITA network → Next.

3

Set HTTP Action

Method: POST  |  URL: http://20.2.0.1:8090/login.xml
Body: mode=191&username=ID&password=PASS&producttype=1

Frequently Asked Questions

Is this script safe?
Absolutely. Everything runs locally on your device. No credentials are uploaded to any server. You can inspect every line of the generated script.
Does it work on mobile?
Yes! Android uses MacroDroid and iOS uses the native Shortcuts app. Follow the guide in the Auto-Login section above.
How do I uninstall?
On Windows, double-click uninstall.bat in the downloaded folder. On Android, delete the MacroDroid macro. On iOS, delete the Shortcut automation.
What does Network Persistence do?
It stops Windows from disconnecting your Wi-Fi or Hotspot when the laptop sleeps or the lid is closed — perfect for overnight downloads.
Is there a 1-tap login for phones?
Yes! Log in once and save the password in Chrome (Google Password Manager) or Safari (iCloud Keychain). Future logins will auto-fill in one tap.
Will it work after a password change?
On Windows, re-generate a new ZIP with your new password and re-run setup.bat. On mobile, update the MacroDroid or Shortcuts action manually.

Privacy Policy

Your data is 100% safe — we store nothing.

GITAConnect operates entirely offline in your browser. Here's exactly how it works:

  • No Server Collection: Credentials you enter are never uploaded to any server, database, or third-party service.
  • Local Only: On Windows, your credentials are saved inside config.json in your local AppData directory, used only by your Windows Task.
  • Mobile Safe: On mobile, we guide you to use native tools (iOS Shortcuts / MacroDroid). No logins are ever stored on this website.
  • Open Source: Every line of the generated scripts can be inspected before running.
